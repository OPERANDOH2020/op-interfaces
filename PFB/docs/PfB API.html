get /getOfferStatus
(getOfferStatusGet)
Get offer status

Query parameters

offer_id (required)
Query Parameter — Unique identifier for an offer

Return type

Status

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Get offer status (active | inactive)

500

Unexpected error
Up 
get /getOffers
Get website offers (getOffersGet)
Extension sends request to PfB service to get a list of offers for the website when signup page is detected

Query parameters

user_id (required)
Query Parameter — Unique identifier representing a specific user from the UDE service
website (required)
Query Parameter — Website address

Return type

array[Offer]

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

An array containing all available offers for the requested website
Up 
get /getPendingDeals

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
